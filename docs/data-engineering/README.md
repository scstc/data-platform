## 一图速览：LLM 数据工程生命周期

```
数据接入/采集 → 数据清洗 → 数据去重 → 质量评估 → 合成与增强 → 数据标注 → 数据集管理
   (S3/DB/API)   (格式/噪声)  (精确/模糊/语义) (打分/过滤)  (造数据/蒸馏)  (人工/智能)  (元数据/版本/血缘)
                              ↑________________ 数据安全与合规贯穿全程 ________________↑
```

## 阅读须知

- **时效性**：本领域 2024-2026 迭代极快，工具的算子数量、架构（如 NeMo Curator 25.09 起转 Ray）都是时间快照，以官方 live 文档为准
- **诚实标注**：证据薄弱的部分（数据投毒防御细节、数据集治理工具实践、标注工具对比）在对应文档里明确标注了「待补研」，不假装全知
- 调研中**被事实核验驳回**的流行说法已剔除，详见各文档的「注意」框
